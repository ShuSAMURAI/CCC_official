Brew Guide

V60 基本レシピ

シンプルで再現性の高いレシピ。ここから好みに合わせて調整できます。

ベースレシピ

  • 豆：15g（中挽き）
  • 湯温：92℃
  • 抽出量：230ml
  • トータル時間：2分30秒

00:00〜30s：30mlで蒸らし / 30s〜1:00：70ml / 1:00〜1:45：80ml / 1:45〜2:10：50ml。

ポイント

  • フィルターは事前にリンスして温める
  • 中心から円を描き、やさしく注ぐ
  • サーバーを軽くスワールして均一化
  • 最後の一滴を落とし切らず、香りを残す

味わいの調整ヒント

酸味・苦味・濃さの調整用メモです。好みに合わせてカスタマイズしてください。

気になる点 調整アイデア
酸味が強い 挽目を細かくする / 湯温を1℃下げる / 1投目をやや増やす
苦味が強い 挽目を粗くする / 抽出量を増やす / 後半の注湯を穏やかに
味が薄い 粉量を+1g / 抽出量を-10〜15ml / 抽出時間を少し延長
味が濃い 粉量を-1g / 抽出量を+10〜15ml / 抽出時間を短く
